# Health OS - Keyboard Shortcuts

## Global Shortcuts

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Cmd/Ctrl + K` | Open Search | Quick search across all content |
| `Cmd/Ctrl + /` | Show Shortcuts | Display this shortcuts panel |
| `Esc` | Close Modal | Close any open modal or dialog |
| `?` | Help | Show help documentation |

## Tips

### Combo Shortcuts
Some shortcuts work in combination:
- Press `G` then another key for navigation (e.g., `G` + `D` for Dashboard)
- Hold `Shift` to modify behavior (e.g., `Shift + Z` for zoom out)

### Context-Specific
Some shortcuts only work in specific contexts:
- Report shortcuts work when viewing reports
- Timeline shortcuts work in timeline view
- Editing shortcuts work in forms and text fields

### Customization
You can customize shortcuts in Settings → Keyboard Shortcuts (coming soon)

### Accessibility
All functionality is available without keyboard shortcuts for accessibility

## Platform Differences

### macOS
- Use `Cmd` key for shortcuts
- Example: `Cmd + K` for search

### Windows/Linux
- Use `Ctrl` key for shortcuts
- Example: `Ctrl + K` for search

## Learning Mode

Enable "Learning Mode" in settings to see keyboard shortcuts as tooltips when you hover over buttons and actions.

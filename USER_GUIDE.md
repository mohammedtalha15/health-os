## Uploading Reports

### Supported File Types

- **PDF** - Lab reports, prescriptions, discharge summaries
- **Images** - JPG, PNG (photos of paper reports)
- **Documents** - DOCX (typed reports)

**Maximum file size:** 10MB per file

### How to Upload

1. Click the **Upload** button in the navigation bar
2. Choose your file:
   - Click to browse your computer
   - Or drag and drop the file
3. Add a **title** (e.g., "Blood Test - Jan 2024")
4. Select **report type**:
   - Lab Report
   - Prescription
   - Discharge Summary
   - Imaging Report
   - Other
5. Click **Upload**

### What Happens Next?

1. **Upload** - File is securely uploaded (a few seconds)
2. **Processing** - AI extracts text and metrics (30-60 seconds)
3. **Analysis** - Metrics are analyzed and compared to reference ranges
4. **Ready** - You'll see a notification when complete

### Tips for Best Results

‚úÖ **Do:**
- Use clear, high-quality scans
- Ensure text is readable
- Upload complete reports
- Use descriptive titles

‚ùå **Avoid:**
- Blurry or dark images
- Handwritten notes (if possible)
- Partial or cropped reports
- Files over 10MB

## Understanding Your Dashboard

### Dashboard Overview

Your dashboard shows:

**Health Score**
- Overall health indicator (0-100)
- Based on your recent metrics
- Color-coded: Green (good), Yellow (caution), Red (attention needed)

**Recent Reports**
- Last 5 uploaded reports
- Processing status
- Quick actions (view, download, delete)

**Key Metrics**
- Important health values
- Trend indicators (‚Üë improving, ‚Üì declining, ‚Üí stable)

**Activity Feed**

### Understanding Metrics

Each metric shows:
- **Current Value** - Your latest reading
- **Unit** - Measurement unit (mg/dL, g/dL, etc.)
- **Reference Range** - Normal range for your age/gender
- **Status** - Color-coded severity:
  - üü¢ Green: Within normal range
  - üü° Yellow: Slightly outside range
  - üî¥ Red: Significantly outside range
- **Trend** - Change from previous reading
- **Explanation** - Plain English description

## Sharing Reports

### Generate Share Link

1. Open a report
2. Click **Share** button
3. Choose options:
   - **Expiration** - 1 day, 1 week, 1 month, never
   - **Password** - Optional password protection
   - **View only** - Prevent downloads
4. Copy link and share

### Email Reports

1. Open a report
2. Click **Email** button
3. Enter recipient email
4. Add message (optional)
5. Send

### Download Reports

Download in multiple formats:
- **Original** - As uploaded
- **PDF** - Formatted report
- **CSV** - Metrics data
- **JSON** - Raw data

## Troubleshooting

### Upload Issues

**Problem:** Upload fails
- Check file size (<10MB)
- Verify file type (PDF, JPG, PNG, DOCX)
- Try different browser
- Check internet connection

**Problem:** Processing takes too long
- Large files take longer
- Complex reports need more time
- Check status in dashboard
- Contact support if >5 minutes

### Display Issues

**Problem:** Metrics not showing
- Refresh the page
- Clear browser cache
- Try different browser
- Check if report processed successfully

**Problem:** Charts not loading
- Disable ad blockers
- Enable JavaScript
- Update browser
- Try incognito mode
